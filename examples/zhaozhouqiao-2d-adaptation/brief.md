# Brief

## 项目基础

- 项目名：`赵州桥·八仙试桥`
- 平台：`16:9 横版主片，可二次裁切为短视频版本`
- 时长：`180秒`
- 语言：`中文`
- 交付物：`2D 动画短片前期生产包`

## 目标

- 业务目标：把赵州桥传说改编成一支可进入生产的 2D 国风动画短片方案
- 受众：`大众文化传播 / 文旅内容受众 / 学生家庭向观众`
- 核心信息：`赵州桥不仅是工程奇迹，更是匠心、民心与燕赵风骨的传奇`
- 希望观众做什么：`记住赵州桥故事，并对其文化价值产生兴趣`

## 创意方向

- 题材：`神话 / 民间故事 / 人文传奇`
- 语气：`传奇、厚重、温热，带一点民间口述感`
- 参考作品：`国风 2D 手绘动画、民间传说短片、壁画感叙事影像`
- 必须出现：`鲁班、张果老、柴王爷、赵州桥、日月五岳、桥上仙迹、百姓见证`
- 必须避免：`写实 3D 质感、纯搞笑化处理、过度仙侠偶像风、桥体结构频繁漂移`

## 约束

- 截止时间：`待定`
- 工具限制：`允许 AI 关键帧与图生视频辅助，但最终风格必须保持 2D 动画感`
- 预算限制：`按中等预算前期方案控制，优先选择可复用资产和可拆分镜头`
- 是否有角色一致性要求：`有，鲁班、张果老、柴王爷、白驴、赵州桥必须全片稳定`

## 改编原则

- 保留原稿的传奇主线，不改人物关系与主题方向
- 允许压缩对白，强化旁白和动作表达
- 允许为了 2D 动画生产效率对镜头密度和场面调度做适配
- 允许把桥的神迹呈现做得更图像化，但不改变传说结论

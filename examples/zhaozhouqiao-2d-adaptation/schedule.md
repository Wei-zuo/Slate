# Schedule

## 生产顺序

1. 锁定 `鲁班 / 张果老 / 柴王爷 / 白驴 / 赵州桥` 五套核心设定与结构图
2. 先做镜头 `03、09、10、12` 的关键帧和粗 animatic，验证主风格、高潮镜头和结尾落点
3. 再批量推进镜头 `01-02、04-08、11` 的中低复杂度镜头
4. 最后统一做合成、转场、旁白、音效与片尾题签

## 优先级

- P0：`镜头 09、10、12`，角色设定锁定，赵州桥结构锁定
- P1：`镜头 03、04、05、06、07、08、11`
- P2：`镜头 01、02` 的群像循环细节和片尾包装

## 风险镜头

- `09`：桥下托桥，层次最多，动作和受力最难
- `10`：仙迹留桥，需要既清楚又有传奇感
- `04-05`：仙人出场，容易风格跑偏

## 排期建议

- 第 1 轮：只做风格定稿和 P0 镜头试跑
- 第 2 轮：确认桥体、角色、仙光、河水表现后，批量做其余镜头
- 第 3 轮：统一调整全片色彩、旁白密度和音乐节奏

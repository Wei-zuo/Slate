# Production Packet

## 项目摘要

- 项目：`赵州桥·八仙试桥`
- 当前版本：`2D 国风动画短片前期整合包`
- 目标时长：`180秒`
- 输出媒介：`16:9 横版主片`
- 表达方式：`旁白主导 + 少量角色短对白 + 强动作节点`
- 生产目标：`先完成风格锁定和高潮镜头试跑，再批量推进整片`

## 必读文件

- `adaptation_notes.md`
- `brief.md`
- `story_package.md`
- `art_package.md`
- `storyboard.md`
- `ad_feedback.md`
- `schedule.md`
- `production_todo.md`

## 执行要求

- 必须保持 `2D 动画感`，不允许漂成写实影视或三维游戏过场
- 必须先完成 `03、09、10、12` 四个镜头的风格和动势验证，再决定是否全量生产
- 全片优先采用 `关键帧驱动 + 局部动效 + 合成` 的方式，不默认追求全镜头高复杂运动
- 对白只做关键句，其他信息交给旁白和画面完成
- 河水、云气、群像、金光优先做可复用层，避免每镜从零生成

## 统一约束

- 鲁班全片身份锚点：`壮年、粗布短衫、石茧双手、低重心、坚定眼神`
- 张果老全片身份锚点：`倒骑白驴、布褡裢、宽檐斗笠、戏谑中带审视`
- 柴王爷全片身份锚点：`素朴布衣、老旧独轮车、沉稳厚重`
- 赵州桥全片结构锚点：`青灰石拱、清楚桥栏、稳定桥腹比例`
- 色彩锚点：`青灰石色 + 冷青河水 + 金色仙光 + 少量暖色人间灯火`
- 叙事锚点：`桥是给百姓走的`，任何镜头表现都不能盖过这一人间动机

## 镜头级特别说明

- 镜头 `03`：
  - 这是赵州桥第一次完整亮相，后续所有桥体都以此镜为结构母版
- 镜头 `09`：
  - 全片最高优先级镜头
  - 必须先试三种画法：`写意水墨边缘版`、`清线条厚涂版`、`中间平衡版`
- 镜头 `10`：
  - 仙迹不要只做物件展示，要表现“被力量压出来”的瞬间
- 镜头 `12`：
  - 结尾优先追求定格感和余韵，不追求复杂运动

## 制片判断

- 当前材料已满足 `进入生产试跑` 的最低条件
- 正式全量生产前，仍需先完成 `桥体结构图`、`仙迹定位图` 和 `桥下托桥 key art`

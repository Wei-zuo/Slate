# Story Package

## 一句话概念

一座为百姓而造的石桥，迎来仙家携日月五岳的试炼，一名凡人匠师以血肉之躯托住桥心，让赵州桥从工程奇迹升成千年传奇。

## 改编策略

- 先用 `民生受阻 -> 桥成通途` 建立赵州桥存在的必要性
- 再用 `仙家不信 -> 下桥试验` 把神话冲突拉起来
- 把全片高潮锁死在 `桥心下沉` 和 `鲁班托桥`
- 结尾不用解释太多历史，只用桥上痕迹和一句话收住传奇感
- 全片以 `旁白主导`，角色对白只在关键点出现

## 故事梗概

燕赵大地上，洨河阻断南北，百姓渡河艰难。鲁班感念民生，以惊人的匠艺造出赵州桥，让两岸终于有了通途。

桥成之日，人间欢腾，云端八仙却为之侧目。张果老不信凡间匠艺有此本事，邀来柴王爷，携日月、五岳落到桥头，要亲自试一试这座桥到底能不能担得起天下盛名。

二仙上桥，桥心骤沉，石缝开裂，洨河翻涌。危急时刻，鲁班跃入河中，以双臂托住桥腹，与仙家重量正面相抗。掌印、蹄印、车沟、膝痕与圆坑在震颤中烙进石面，赵州桥终究稳住。张果老与柴王爷被凡人匠魂折服，俯身称桥。从此，赵州桥不仅是桥，更成了燕赵风骨和匠心不朽的象征。

## 创意方向

- 叙事定位：`民间传奇短片`，不是纯历史讲解，也不是轻喜剧神话
- 画面母题：`石与光`
- 戏剧母题：`凡人匠力正面承接仙家神力`
- 表达方式：`大画面 + 少对白 + 强动作节点`
- 适配方式：`按 12 个镜头组织，保证每一镜都能拆给后续分镜和生产`

## 情绪节奏

- `0-36秒`：民生之苦转为桥成之喜，建立赵州桥的价值
- `36-64秒`：云端起意，神话力量逼近，气势抬升
- `64-110秒`：试桥推进，桥身承压，危险不断累积
- `110-148秒`：桥心惊变，鲁班托桥，进入全片最高峰
- `148-180秒`：仙凡和解，桥上留痕，传奇定格

## 角色与目标

- 鲁班：目标是 `护住桥，也护住百姓的通途`。他不是为了赢神仙，而是为了不让桥塌。
- 张果老：目标是 `验真`。一开始带戏谑和不信，结尾转为敬佩。
- 柴王爷：目标是 `辨值`。他更沉稳，代表对“这桥是否真担得起天下第一” 的判断。
- 百姓：目标是 `盼桥稳、盼路通`。他们是鲁班行为的情感依据，也是全片的人间温度。

## 旁白 / 对白草案

- 旁白：`洨河横亘燕赵，南北往来，苦在一水之间。`
- 旁白：`鲁班造桥，不为惊天，只为让百姓脚下，有一条稳路。`
- 张果老：`人间一夜成桥？我倒要看看，这石桥到底有多硬。`
- 柴王爷：`若真能承此分量，便当得起天下盛名。`
- 鲁班：`桥为百姓所造，莫说二位，千军万马过，也该稳。`
- 旁白：`日月压桥，五岳坠心，石缝如雷，洨河翻沸。`
- 鲁班：`桥，不能塌！`
- 张果老：`好个鲁班！好座赵州桥！`
- 柴王爷：`能承日月、载五岳，当为天下第一桥。`
- 鲁班：`心诚，桥便坚。`

## 给美术设计的提示

- 视觉不要走飘逸仙侠，而要走 `石头有分量、河水有寒意、仙光有压迫感`
- 鲁班必须像 `干过重活的人`，手、背、肩、脚下重心都要能看见
- 张果老和柴王爷要有仙家身份，但不能脱离民间传说语境
- 赵州桥是全片真正主角之一，桥体结构必须稳定、可信、可重复绘制
- `石与光` 是主母题，青灰石和金色天光的反差必须成立

## 给副导演的提示

- 前半段镜头可以更平稳，越靠近桥心惊变，镜头越要收紧、加压
- `桥心下沉` 和 `鲁班托桥` 必须给足时长，不要一笔带过
- 百姓不是背景板，至少要有两次明确的群体反应
- 仙迹留桥需要做成 `可被观众记住的镜头链`
- 结尾不要拖长，用一个稳镜和一句收束台词完成传奇落点
